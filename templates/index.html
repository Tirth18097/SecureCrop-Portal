Cryptography Stack

Three-Layer Security Model

Every piece of crop data is protected by three independent cryptographic layers working together.

🔒
Confidentiality
AES Encryption

Advanced Encryption Standard encrypts crop data so only authorized officers with the correct key can read it. Uses CBC mode for maximum security.

CBC Mode · 16-byte key
🔑
Integrity
SHA-256 Hashing

Cryptographic hash detects any tampering. If even a single character changes, the hash value will be completely different — instantly alerting to tampering.

256-bit digest · One-way
✍️
Authentication
RSA Authentication

Digital signatures prove data originated from the authentic farmer. The department verifies identity using the public key before accessing any record.

PKCS#1 v1.5 · 2048-bit
